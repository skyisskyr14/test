# 面向中小企业的员工绩效考核管理系统 - 运行说明

## 目录结构
- `spring/`：Spring Boot + MyBatis 后端工程
- `web/`：Vue3 + Element Plus 前端工程
- `data/`：数据库 SQL 初始化脚本
- `output/`：一键运行脚本与 Docker Compose 编排

## 当前已完成功能（优先功能健全）
1. 认证登录 + 会话 Token
2. 考核方案管理（新建、发布）
3. 方案指标分配（绑定指标及权重）
4. 方案参与人分配（自动生成任务）
5. 指标库管理（新增、删除、列表）
6. 员工任务中心（查看任务）
7. 员工自评打分（按指标提交得分）
8. 自动核算结果（生成等级 A/B/C/D）
9. 管理层结果审批（通过/驳回）
10. 绩效申诉（提交、处理）
11. 报表页（部门对比、趋势）

## 方式一（推荐）：Docker 一键启动
```bash
cd output
./run.sh
```
前端：`http://localhost:5173`  后端：`http://localhost:8080/api`

停止：
```bash
cd output
./stop.sh
```

## 方式二：本地启动
1. 导入数据库：执行 `data/init.sql`
2. 启动后端：
```bash
cd spring
mvn spring-boot:run
```
3. 启动前端：
```bash
cd web
npm install
npm run dev
```
