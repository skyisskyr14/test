# 面向中小企业的员工绩效考核管理系统 - 运行说明

## 目录结构
- `spring/`：Spring Boot + MyBatis 后端工程
- `web/`：Vue3 + Element Plus 前端工程
- `data/`：数据库 SQL 初始化脚本
- `output/`：一键运行脚本与 Docker Compose 编排

## 当前已完成功能（本次升级）
1. **认证与权限**：登录后签发 Token；后端拦截器保护全部业务接口；基础角色权限校验（HR/管理层）
2. **考核方案管理**：方案列表、新建方案
3. **指标库管理**：指标列表、新增、删除
4. **员工任务中心**：我的考核任务查询、自评提交
5. **结果审批**：管理层审批通过/驳回
6. **绩效申诉**：员工提交申诉、HR处理申诉
7. **数据分析**：部门绩效对比、周期趋势

## 方式一（推荐）：Docker 一键启动
1. 安装 Docker 与 Docker Compose
2. 在项目根目录执行：
   ```bash
   cd output
   ./run.sh
   ```
3. 访问：
   - 前端：`http://localhost:5173`
   - 后端接口：`http://localhost:8080/api`

> 演示账号：
> - HR：`hr_admin / 123456`
> - 管理层：`leader / 123456`
> - 员工：`employee / 123456`

停止服务：
```bash
cd output
./stop.sh
```

## 方式二：本地分别启动
### 1) 初始化数据库
将 `data/init.sql` 导入你的 MySQL（账号 root/root，库名 perf_system）。

### 2) 启动后端
```bash
cd spring
mvn spring-boot:run
```

### 3) 启动前端
```bash
cd web
npm install
npm run dev
```
